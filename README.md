# Urban Road Accident Risk Prediction

## Project Goal
Predict high-risk urban road segments using traffic, weather, and road conditions.

## Folder Structure
- `data/` : datasets (sample pushed, full dataset local)
- `notebooks/` : Jupyter notebooks for preprocessing, EDA, and modeling
- `results/` : plots and outputs

## Workflow
1. Data Preprocessing
2. EDA & Visualization
3. Predictive Modeling

## How to Run
1. Clone repo: `git clone <your-repo-url>`
2. Install requirements: `pip install -r requirements.txt`
3. Run notebooks in order

## Notes
- Full dataset is too large for GitHub; see Kaggle link for full data.
